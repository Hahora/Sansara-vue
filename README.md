# Sansara Mini App Frontend

Это фронтенд для Telegram Mini App бального комплекса Сансара, созданный с использованием Vue 3, Vite, Tailwind CSS и Pinia.

## Обзор

Sansara Mini App Frontend предоставляет пользовательский интерфейс для взаимодействия с бэкендом API, позволяя пользователям просматривать программы, мероприятия, управлять профилем и создавать бронирования через веб-интерфейс внутри приложения Telegram.

## Особенности

- **Vue 3 Composition API**: Современная архитектура компонентов
- **Pinia Store**: Централизованное управление состоянием
- **Tailwind CSS**: Адаптивный и отзывчивый дизайн
- **Vue Router**: Навигация между страницами
- **Интеграция с API**: Взаимодействие с бэкендом Sansara MiniApp API
- **Интеграция с Telegram WebApp**: Использование SDK для аутентификации и получения данных пользователя

## Установка

1. Установите зависимости:
   ```bash
   npm install
   ```

2. Настройте переменные окружения в `.env.local` (на основе `.env.example`):
   ```
   VITE_API_URL=ваш_адрес_api
   ```

3. Запустите режим разработки:
   ```bash
   npm run dev
   ```

## Сборка

Для сборки проекта в продакшен:
```bash
npm run build
```

Для локального просмотра продакшен-сборки:
```bash
npm run preview
```

## Архитектура

Фронтенд следует модульной архитектуре с:

- **Components**: Переиспользуемые UI-компоненты
- **Pages**: Страницы приложения
- **Stores**: Хранилища Pinia для управления состоянием
- **Utils**: Вспомогательные функции и API-клиент
- **Assets**: Стили, изображения и другие ресурсы
- **Router**: Маршруты приложения

## Интеграция с API

Фронтенд взаимодействует с бэкендом через API-клиент, расположенный в `src/utils/api.js`. Все запросы автоматически включают данные аутентификации Telegram WebApp, если они доступны.

## Интеграция с Telegram WebApp

Приложение использует Telegram WebApp SDK для:
- Аутентификации пользователей
- Получения данных пользователя
- Адаптации интерфейса под Telegram

## Стили

Проект использует Tailwind CSS для стилизации с минималистичным дизайном, соответствующим требованиям Telegram Mini Apps.

## Роутинг

Приложение использует Vue Router для навигации между страницами. Все основные разделы бота представлены как отдельные страницы:
- Главная
- Профиль
- Программы (коллективные и авторские)
- Мероприятия
- Система лояльности
- Подарочные сертификаты
- Бронирование
- И другие

## Управление состоянием

Pinia используется для централизованного управления состоянием приложения, включая:
- Данные пользователя
- Списки программ и мероприятий
- Состояние бронирований
